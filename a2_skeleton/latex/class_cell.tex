\doxysubsection{Detailed Description}
Cells will be used as areas of space (rectangles) where the sensor data will be fused to indicate occupancy, default size provided, centre location draw randonly from a map of max size. On creation state is U\+N\+K\+N\+O\+WN. 

The \mbox{\hyperlink{class_cell}{Cell}} will be used for fusion accordinly\+: ~\newline
If the sensor intersects the cell and goes through \mbox{\hyperlink{class_cell}{Cell}} will be F\+R\+EE~\newline
If the sensor has a return from within \mbox{\hyperlink{class_cell}{Cell}}, it will be O\+C\+C\+U\+P\+I\+ED~\newline
